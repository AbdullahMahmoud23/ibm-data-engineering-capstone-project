# Module 4: Data Analytics and Business Intelligence

## Project Overview

In the final analytical phase of the pipeline, I transitioned from data engineering to data visualization. Using **Google Looker Studio**, I connected to the extracted e-commerce dataset to build an interactive Business Intelligence dashboard. The goal of this module was to transform raw sales records into actionable visual insights for stakeholders.

## Data Sources

The sample data for this project was provided in CSV format. You can download the raw dataset used to populate the data Analytics here:

- [ecommerce.csv](https://cf-courses-data.s3.us.cloud-object-storage.appdomain.cloud/IBM-DB0321EN-SkillsNetwork/analytics/ecommerce.csv)

## Tools & Technologies

- **Google Looker Studio:** For building the interactive BI dashboard.
- **CSV:** Used as the flat-file data source extracted from the upstream Data Warehouse.

## Visualizations & Business Insights

I developed several key reporting metrics to analyze sales performance across different dimensions:

1. **Sales Trend Analysis (Line Chart):** \* **Metric:** Total Sales
   - **Dimension:** Month
   - **Filter:** Year 2020
   - _Insight:_ Tracks the month-over-month revenue performance to identify seasonal peaks and drops throughout the year 2020.

2. **Revenue Distribution (Pie Chart):**
   - **Metric:** Total Sales
   - **Dimension:** Category
   - _Insight:_ Illustrates the proportion of total revenue generated by each product category (e.g., Mobile Phones, Movies, Ebooks).

3. **Quarterly Product Performance (Bar Chart):**
   - **Metric:** Total Sales
   - **Dimension:** Quarter
   - **Filter:** Category = 'Mobile Phone'
   - _Insight:_ Isolates a specific high-value category to evaluate its quarterly growth and sales velocity.
